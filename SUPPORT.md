# Support

## Getting Help

This is a community-maintained project. For support:

1. **GitHub Issues** — Search existing issues or open a new one for bugs and feature requests
2. **Documentation** — See [README.md](README.md) for installation and configuration guides
3. **AlertLogic API** — For questions about the AlertLogic API itself, refer to the [AlertLogic Developer documentation](https://docs.alertlogic.com/)

## Reporting Security Issues

Please do not report security vulnerabilities through public GitHub issues. Instead, open a private security advisory in the GitHub repository.

See [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines on contributing to this project.
